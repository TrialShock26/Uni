3.4 - Già espresso come chiave primaria

Si scriva in algebra relazionale una espressione che restituisce gli automi che riconoscono lo
stesso insieme di parole dell’automa di codice XXX.


S1 (CodS1, CodA1, In1, Fin1) = Stato (CodS, CodA, Iniziale, Finale)
S2 (CodS2, CodA2, In2, Fin2) = Stato (CodS, CodA, Iniziale, Finale)
R0 = S1 X S2
R1 = Raggiungibile JN CodS1=Start ^ In1=TRUE ^ CodS2=End ^ Fin2=TRUE ^ CodA1=CodA2 ^ CodA1=CodA R0
X = PR CodA, Parola (R1)
Y (CodA, Parola1) = SL CodA="XXX" (X)
YCount (NParole) = F COUNT Parola1 (Y)
Z = PR CodA (X JN Parola!=Parola1 Y)
A = (PR CodA (Automa))-Z
A' = A * R1
ACount (CodA, NParole) = CodA F COUNT Parola (A')
Result = PR CodA (Acount * YCount)
